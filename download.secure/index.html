todaystime software download

First, download xampp here

Install to C:\todaystime-webserver\xampp

Copy and paste your HTML documents to xampp\htdocs, removing the existing files beforehand

Open xampp-control.exe and start Apache (and MySQL for phpmyadmin)

Your website will be hosted at http://localhost/yourhtmldoc.html

If you have a folder in htdocs, like htdocs\subsite with index.html in it, you can go to http://localhost/subsite without "index.html"

Folders containg index.html will automatically use index.html when visiting the folder with localhost.

For help, stylesheets, open-source code, security guarantees, etc. visit todaystime or see the next section

Starting MySQL will allow you to go to http://localhost/phpmyadmin

Setting up todaystime on localhost

Download all todaystime HTML files and add them to the root directory of htdocs

Make a new folder in the root directory of htdocs with the name of your site and cut and paste your HTML docs into that folder

Then todaystime will be setup at localhost, so you can access your website and get help through todaystime

This is optional, but is reccommended if you would like help
